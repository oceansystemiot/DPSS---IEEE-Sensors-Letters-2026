##This repository contains the data and/or code associated with the following article.

- **Title**: Design and Towing-Tank Validation of a Distributed Pressure Sensor System for AUV Drift-Angle and Speed Estimation: A Preliminary Comparative Study
- **Authors**: Jin-Yeong Park et al.
- **Journal/Publication**: IEEE Sensors Letters

## Description

hydro_dynamic_deltaP_extended_reduced.xlsx
: deltaP vector of each experiment (3~36, except 21~24)

Sparus2head profile.xlsx
: Sparus-II AUV head profile with respect to x-y plane with unit:mm.

Pdynamic.xlsx
: Time-series of every nine pressure sensors for every experiments (3~36, except 21~24.)
